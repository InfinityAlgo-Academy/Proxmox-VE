# Network Configuration

## Configure Network

```bash
# Edit /etc/network/interfaces
nano /etc/network/interfaces

# Restart networking
systemctl restart networking
```

## Network Commands

| Command | Description |
|---------|-------------|
| `ip addr` | Show addresses |
| `ip link` | Show interfaces |
| `ip route` | Show routes |
