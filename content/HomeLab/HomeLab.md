# Home Lab

## Recommended Setup

| Component | Specification |
|-----------|----------------|
| CPU | Intel i5/i7 or Ryzen 5/7 |
| RAM | 32-64 GB |
| Storage | 2-4 TB |
| Network | 1 Gbps |

## Proxmox on Home Lab

- Run nested virtualization
- Test clustering
- Practice backup/restore

## Services to Run

- Docker
- Home Assistant
- Pi-hole
- Jellyfin
- AdGuard Home
