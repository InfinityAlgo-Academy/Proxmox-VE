# Backup Strategies Guide

## Types of Backups

1. **Full Backup** - All VM/CT data
2. **Incremental** - Changed data only
3. **Differential** - Changes since last full

## 3-2-1 Rule

- 3 copies of data
- 2 different storage types
- 1 offsite backup

## Configuration

```bash
# Schedule backup job
# Datacenter → Backup → Add
```
