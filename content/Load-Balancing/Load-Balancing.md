# Load Balancing

## HAProxy

```bash
# Install HAProxy
apt install haproxy -y

# Configure
nano /etc/haproxy/haproxy.cfg
```

## Nginx

```bash
# Install Nginx
apt install nginx -y

# Configure load balancing
nano /etc/nginx/nginx.conf
```
