# How to Use Container Templates - Complete Guide

## Question: How do I use new container templates?

### Answer: Template management

---

## How to Download New Templates

### Step 1: Browse Available
```bash
# Update template list
pveam update

# List available
pveam available
```

---

## How to Import Custom Template

### Step 1: Import
```bash
# Import local template
pveam import local debian-12 debian-12.tar.gz vztmpl
```

---

## How to Update Templates

### Step 1: Update All
```bash
# Update all templates
pveam update
```
