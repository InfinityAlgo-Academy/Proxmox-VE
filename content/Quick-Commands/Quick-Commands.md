# Quick Commands

## VMs

| Command | Description |
|---------|-------------|
| `qm list` | List VMs |
| `qm start 100` | Start VM |
| `qm stop 100` | Stop VM |
| `qm console 100` | Console |
| `qm migrate 100 node2` | Migrate |

## Containers

| Command | Description |
|---------|-------------|
| `pct list` | List containers |
| `pct start 200` | Start |
| `pct stop 200` | Stop |
| `pct exec 200 --` | Execute |

## Storage

| Command | Description |
|---------|-------------|
| `pvesm status` | Storage list |
| `pvesm add dir local` | Add storage |
| `pvesm scan-lvm` | Scan LVM |

## Cluster

| Command | Description |
|---------|-------------|
| `pvecm status` | Cluster status |
| `pvecm nodes` | List nodes |
| `ha-manager status` | HA status |
