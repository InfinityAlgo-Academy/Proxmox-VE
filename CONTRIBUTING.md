# Contributing to Proxmox VE Knowledge Base

Thank you for your interest in contributing!

## How to Contribute

### 1. Add New Articles

1. Fork the repository
2. Create a new branch: `git checkout -b new-article`
3. Add your article in the appropriate folder
4. Submit a pull request

### 2. Article Guidelines

- Use clear, concise language
- Include command examples with output
- Add "See Also" links to related articles
- Use frontmatter with title:

```markdown
---
title: Your Article Title
---

# Your Article Title

## Overview

...

## Commands

\`\`\`bash
# your commands here
\`\`\`

## See Also

- [[index|Back to Proxmox VE]]
```

### 3. Editing Existing Articles

- Keep formatting consistent
- Test commands before submitting
- Add relevant links to other articles

## Questions?
